Instruções:
	1. Exporte o projeto para sua IDE de preferência.
	2. Digite uma entrada de testes segundo o exemplo abaixo.

Exemplo de entrada para testes:

5 5
1 2 N
LMLMLMLMM
3 3 E
MMRMMRMRRM

Saída esperada:
1 3 N
5 1 E

-------
ONDE:
5 5  = tamanho do terreno (platô)
1 2 N = orientação da sonda
LMLMLMLMM = comandos
